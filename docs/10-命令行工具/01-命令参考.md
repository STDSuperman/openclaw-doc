# 命令行工具参考

## Agent 命令

### 运行 agent

```bash
openclaw agent --message "你好,请帮我生成周报"
```

选项:
- `--message`: 必需,要发送给 agent 的消息
- `--to`: 目标地址(电话号码、组 ID 等)
- `--session-id`: 重用特定会话
- `--agent`: 使用特定 agent
- `--thinking`: 思考级别
- `--verbose`: 详细级别
- `--deliver`: 投递回复
- `--local`: 本地运行
- `--json`: JSON 输出

### 通过 CLI 直接发送

```bash
openclaw message send --to +15551234567 --message "测试消息"
```

## Gateway 命令

### 运行 Gateway

```bash
openclaw gateway --port 18789
```

选项:
- `--port`: Gateway 端口(默认 18789)
- `--bind`: 绑定地址(loopback 或 0.0.0.0)
- `--mode`: 运行模式(local 或 gateway)
- `--verbose`: 详细输出

### 配置管理

```bash
openclaw config get gateway.port
openclaw config set gateway.port 8080
openclaw config unset gateway.port
```

### 安装 Gateway 服务

```bash
openclaw gateway install
```

### 重启 Gateway

```bash
openclaw gateway restart
```

## 全局选项

### JSON 输出

大多数命令支持 `--json` 标志以输出结构化数据

### 详细模式

使用 `--verbose` 或 `-v` 启用详细输出

### 配置文件

通过环境变量或 `--config-path` 指定自定义配置文件位置

## 使用技巧

1. **Tab 补全**: 所有命令都支持 Tab 补全
2. **历史记录**: 上下箭头查看和重复命令
3. **快速别名**: 使用别名来加速常用操作
4. **管道操作**: 可以将命令输出管道到其他命令
5. **批处理**: 在脚本中使用命令进行批量操作
