## 术语规范

### 技术术语

首次出现技术术语时提供中英文对照：

- **Gateway（网关）**：OpenClaw 的核心控制平面
- **Session（会话）**：与 AI 的对话上下文
- **Node（节点）**：连接到网关的设备

后续出现时使用简称。

### 命令保留

CLI 命令和配置键保留英文：

```bash
openclaw gateway --port 18789
```

不翻译为：

```bash
openclaw 网关 --端口 18789
```
